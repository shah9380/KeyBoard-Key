
# KeyBoard Keycode Viewer

[Feel Free to Explore](https://shah9380.github.io/KeyBoard-Key/)

This is a simple web application that displays keyboard keycodes when you press a key on your keyboard. It provides a clear visual representation of the key pressed and its corresponding keycode.

## Features

- Dynamic display: The application updates in real-time as you press different keys, showing you the pressed key and its keycode.
- User-friendly interface: The interface is designed to be easy to use and visually appealing.
- Green key highlight: The key you press is highlighted in green for better visibility.

## How to Use

1. Open the web page in your browser.
2. Press any key on your keyboard.
3. The pressed key will be displayed in a larger font in green, along with its numeric keycode.
